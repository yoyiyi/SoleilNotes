## [题目](https://leetcode-cn.com/problems/single-number/)

难度：简单

给定一个**非空**整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

**说明：**

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

**示例 1:**

```
输入: [2,2,1]
输出: 1
```

**示例 2:**

```
输入: [4,1,2,1,2]
输出: 4
```

## 解法

* 任何数和 0 做异或运算，结果仍然是原来的数
* 任何数和其自身做异或运算，结果是 0
* 异或运算满足交换律和结合律

```java
class Solution {
    public int singleNumber(int[] nums) {
        int single = 0;
        for(int n : nums){
            single ^= n;
        }
        return single;
    }
}
```
